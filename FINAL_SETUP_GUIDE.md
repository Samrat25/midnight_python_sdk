## 🔧 Troubleshooting

### Services Won't Start

**Problem**: Docker container conflicts

**Solution**:
```bash
python start_clean.py
```

This will:
- Clean all Docker containers
- Remove conflicts
- Start services fresh

### Services Show OFFLINE

**Problem**: Services still starting or crashed

**Solutions**:

1. **Wait longer** - Services take 1-2 minutes to fully start
   ```bash
   midnight-py status
   ```

2. **Check logs** in the midnight-local-dev window

3. **Restart services**:
   ```bash
   cd ../midnight-local-dev
   npm start
   ```

### Wallet Has No DUST

**Problem**: Balance shows 0

**Solution**:
1. Go to midnight-local-dev window
2. Choose option [1]
3. Your mnemonic is already configured
4. Wait for funding to complete

### Can't Find midnight-local-dev Window

**Problem**: Window closed or hidden

**Solution**:
```bash
cd ../midnight-local-dev
npm start
```
